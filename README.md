# Music Toolbox

A collection of small, standalone web apps for music theory.

## Usage

Open any `.html` file directly in your web browser. No server or build step required.

## Tools

- **Scale Helper** (`scale-helper.html`) - A reference for musical scales
- **Key Finder** (`key-finder.html`) - Suggests possible keys from a song's constituent chords
- **Strumming Pattern** (`strumming-pattern.html`) - Build and play back rhythmic strumming patterns
